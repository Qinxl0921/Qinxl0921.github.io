Nacos注册中心

2024/12/28 微服务

Nacos官方文档：https://nacos.io/docs/v2.3/quickstart/quick-start/?spm=5238cd80.216780ba.0.0.4dbbcd368esDKS

1、配置MySQL

新建一个nacos数据库，主要用于nacos的持久化。

2、使用docker部署

配置文件：custom.env

用于配置Nacos和MySQL连接信息。

docker部署命令：

docker run -d \
--name nacos \
--env-file ./nacos/custom.env \
-p 8848:8848 \
-p 9848:9848 \
-p 9849:9849 \
--restart=always \
--network qmall-net \
nacos/nacos-server:v2.1.0-slim

访问nacos：http://IP:8848/nacos

如果用的是云服务器，记得在云服务器控制界面，添加防火墙规则！

3、服务注册

1）引入nacos discovery依赖

<!--nacos 服务注册-->
<dependency>
    <groupId>com.alibaba.cloud</groupId>
    <artifactId>spring-cloud-starter-alibaba-nacos-discovery</artifactId>
</dependency>

2）配置nacos地址

在application.yml文件里配置nacos地址：

cloud:
    nacos:
        server-addr: ip:8848

启动服务后，就可以在nacos管理页面->服务管理->服务列表中查看注册的服务。

4、服务发现

服务注册完成后，使用DiscoveryClient进行服务调用：

private final DiscoveryClient discoveryClient;
private void handleCartItems(LIst<CartVO> vos){
    // 根据服务名称，拉取服务的实例列表
    List<ServiceInstance> instances = discoveryClient.getInstances("xxx-service");
}

这样调用服务的方式太麻烦了，后面可以使用OpenFeign。
