如何使用Hexo搭建个人博客

2024/12/22 guide

1. 修改博客默认配置

可以打开Blogs文件夹下面的_config.yml文件，在里面修改博客的默认标题，作者等配置。

## Site
title: 标题
subtitle: 副标题
description: 描述
keywords: 关键词
author: 站主
language: 语言（可以填写zh-CN）
timezone: 时区（可以填写Asia/Shanghai）

修改完配置文件后，使用如下命令，更新配置文件：

npx hexo g	# 生成博客
npx hexo d	# 上传

npx用于临时执行hexo

注意每次更改后，博客绑定域名都会恢复成原始的，需要重新绑定。

2. 书写并上传博客

在Blogs文件夹下面打开git bash，使用如下命令创建新的博客：

npx hexo new '文章标题'

运行此命令，会在Blogs/source/_posts文件夹下面生成以文章标题为文件名的md文件。然后使用Typora或者语雀打开，即可编写博客内容。

写好博客后，在git bash中使用如下命令生成并上传博客：

npx hexo g	# 生成博客
npx hexo d	# 上传博客

3. 文章缩略

缩略展示文档内容有两种方法：

  1. 使用语法

    在文档内容中使用，这样在网页上只能预览到之前的内容，后面的内容需要点击阅读全文才能查看。

  2. 在Front-Matter里设置description

    即在文档开头的description参数下设置想要预览的内容，此内容不会出现在正文中。

    description: |
    文档预览内容，不会出现在正文中。
